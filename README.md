# JDFC

## Requirements

### Experiment
#### General
```
- Defects4j
- Bash 4 or higher
- Python 3
```

#### Bash
```
- xmlstarlet
- jq
```

#### Python
```
- pip3
- regex
```

## Run Branch
```
cd <path-to-jdfc>/experiment
/opt/homebrew/bin/bash run_branch.sh <temp-defects4j-repo-folder>
```

## Run Dataflow
```
cd <path-to-jdfc>/experiment
/opt/homebrew/bin/bash run_dataflow.sh <temp-defects4j-repo-folder>
```
